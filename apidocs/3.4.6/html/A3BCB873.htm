TypedCollectionResolverT Class
A collection resolver based of the default CollectionResolver in Castle Windsor but it only resolves collections of the specified type (T) Use it by adding it to the container at wire-up with container.Kernel.Resolver.AddSubResolver(new TypedCollectionResolver<CollectionItemType>(container.Kernel));
