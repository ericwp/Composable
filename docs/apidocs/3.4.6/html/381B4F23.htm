IPersistenceSession Interface

Wraps a number of other interfaces together for convenience. It is generally recommended to use the more fine grained interfaces instead.

Consider using one of these instead: IEntityFetcher, IEntityPersister, IQueryableEntities, IQueryableEntityFetcher
