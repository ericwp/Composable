IdEqualityObjectTEntity, TKEy Class
Base class for any class that considers equality to be that the Ids for two instances are the same. It provides implementations of Equals(Object), GetHashCode, and IEquatableT. Equals is implemented as: return

!ReferenceEquals(null, other) && other.Id.Equals(Id)
the operators simply uses Equals.
